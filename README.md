# 🛠️ Shofyra Admin Panel

Shofyra Admin Panel is a web-based management system designed to control and manage the Shofyra eCommerce mobile application. It allows administrators to handle products, users, and orders efficiently.

## 🚀 Features
- 📦 Product Management (Add, Update, Delete products)
- 👤 User Management
- 🛒 Order Management
- 📊 Dashboard with analytics overview
- 🔐 Admin authentication system
- 🖼️ Image upload for products
- ⚡ Real-time database integration

## 🛠️ Technologies Used
- Node.js / Express.js
- MongoDB (Database)
- Firebase (optional integration)
- HTML, CSS, JavaScript
- REST API

## 📂 Project Structure
- `routes/` - API routes
- `models/` - Database models
- `controllers/` - Business logic
- `config/` - Configuration files
- `public/` - Static assets (images, styles)
- `views/` - UI templates (if applicable)
